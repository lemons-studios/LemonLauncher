## 🛠️ Running from source

### Pre-requisites

- Node.js
- Yarn

### Running Tauri app

```sh
yarn tauri dev
```

## 🚀 Distributing

1. Create a Tauri app package using

```sh
yarn tauri build
```

2. Generated package can be found at `./src-tauri/target/release/Mission Monkey Installer.exe`
